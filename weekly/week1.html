Andrew the Ant Hint 3

The longest time it takes for an ant to walk off the left end is max(x), where x ranges over the starting position of all ants initially facing left. The time for the right side is max(l-x), where x ranges over those ants initially facing right.


Andrew the Ant

Let's solve it for those going left. Let x be the maximum starting position of any ant facing left. Let y be the number of ants to the left of x that are initially facing right. Then the ant that takes x time to walk off the left side will be the ant which starts y positions to the left of x.


DA-Sort

Any item x that has an item y occurring after it with y < x *must* be deleted. Additionally, if x must be deleted, then all elements larger than x must be deleted (as, when x is moved to the end, those become invalid). So, find the smallest x that has a y after it with y < x. Then x and everything larger than x must be deleted. Everything else is safe. There is some edge case here about things equal to x, but I leave that as an exercise to the reader.
